# ShortLink
Сайт для преобразования длинных ссылок в короткие

## Инструкция по использованию сервиса сокращения ссылок
### 1. Доступ к сервису
Перейдите на сайт: https://shortlink-olive.vercel.app/

### 2. Как сократить ссылку
**Шаг 1: Вставьте длинную ссылку**
· Найдите в адресной строке браузера длинную ссылку, которую нужно сократить
· Скопируйте её (Ctrl+C или правой кнопкой мыши → "Копировать")
· Вставьте ссылку в поле ввода на главной странице сайта

**Шаг 2: Сократите ссылку**
· Нажмите кнопку "Сократить"
· Дождитесь обработки — обычно это занимает несколько секунд

**Шаг 3: Получите короткую ссылку**
· Система сгенерирует короткую ссылку 
· Скопируйте её с помощью соответствующей кнопки или вручную

### 3. Дополнительные возможности
Просмотр статистики
· Сокращенные ссылки могут предоставлять статистику переходов

### 4. Использование короткой ссылки
Вставьте скопированную короткую ссылку в нужное место:
Сообщения в мессенджерах
Социальные сети
Презентации
Печатные материалы
При переходе по короткой ссылке пользователь будет автоматически перенаправлен на исходный длинный URL

### 5. Важная информация
Короткие ссылки обычно не имеют срока действия
Не используйте сервис для сокращения ссылок на запрещенный или вредоносный контент
Сервис бесплатен для обычного использования
